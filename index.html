Projects

Aerospace Mechatronics Software/AI Electrical

Autonomous Quadcopter Simulation using MATLAB

This project involved simulating the flight of an autonomous quadcopter using MATLAB and Simulink. The control system was designed using PID and LQR control methods to stabilize the quadcopter and guide it through a pre-defined flight path.

Tilt-Rotor VTOL Simulation in Unreal Engine

Using Unreal Engine, I simulated the flight dynamics of a tilt-rotor VTOL aircraft. This simulation was focused on the transition between vertical takeoff/landing and forward flight, providing insights into control strategies and flight stability.

Flight Path Optimization for UAVs

This project focused on developing algorithms to optimize the flight paths of UAVs in complex environments. By minimizing energy consumption and maximizing flight efficiency, the UAVs were able to traverse various terrains autonomously.

Inverse Kinematics Solver for a 5-DOF Robotic Arm

Developed an inverse kinematics algorithm for a 5-DOF robotic arm. The solver allowed precise control of the robotic arm's end-effector, enabling it to follow complex trajectories in a 3D workspace.

Terrain Modeling using Project Chrono

Using Project Chrono, I modeled different types of terrains and evaluated the interaction between robotic wheels and deformable soil. The project focused on improving the traction of robots in rough terrain environments.

Robotic Arm Path Planning

Designed a path planning algorithm for a 6-DOF robotic arm using the Rapidly-exploring Random Trees (RRT) method. The algorithm allowed the robot to navigate obstacles and complete tasks in cluttered environments.

Reinforcement Learning for Quadcopter Control

This project applied reinforcement learning (RL) techniques to develop a control policy for a quadcopter. The RL model was trained in a simulation environment to minimize errors and stabilize the flight path under different environmental conditions.

Mapping Algorithms for Mower Simulation

Developed a programmable mower simulation using Unreal Engine that utilized mapping and path planning algorithms to autonomously mow a lawn. The system integrated obstacle avoidance and real-time mapping capabilities.

Theremin Build

I designed and built a functioning Theremin for a music and engineering course. The project involved creating the oscillators, antennas, and control circuitry, as well as tuning the instrument to achieve the desired frequency range.

Long-range Rover Antenna Design

This project focused on improving the communication range of a rover by designing and testing antennas operating at 900 MHz and 2.4 GHz. Various antenna designs were simulated and tested to maximize signal strength and communication reliability.
